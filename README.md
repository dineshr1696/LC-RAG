
#  Global GRC RAG Assistant

An AI-powered Retrieval-Augmented Generation (RAG) assistant that answers governance, risk, and compliance (GRC) questions using country-specific policy PDFs.

##  Features
- Multi-PDF ingestion and semantic search
- Retrieval + Generation using `Mistral-7B-Instruct`
- Context-aware factual answers
- Built with LangChain, Pinecone, HuggingFace, and Streamlit

##  Tech Stack
- **Embeddings:** intfloat/e5-base-v2  
- **LLM:** mistralai/Mistral-7B-Instruct-v0.2  
- **Vector DB:** Pinecone  
- **Framework:** LangChain + Streamlit
